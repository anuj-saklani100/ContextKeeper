# ContextKeeper

Multi-model AI chat with persistent context. Switch between Claude, GPT, Gemini, and local models without losing your conversation history.

## Features
- 💬 Chat with Claude, OpenAI, Gemini, or local Ollama models
- 🔄 Switch models mid-conversation — context carries over automatically
- 💾 History saved locally to `~/.contextkeeper/` as markdown files
- 📋 Pick which past sessions to inject when switching models
- 🖱️ Right-click any code → Explain or Ask about it

## Setup
1. Open Settings (`Cmd/Ctrl + ,`) and search **ContextKeeper**
2. Add your API keys (Anthropic / OpenAI / Google)
3. Click the chat icon in the sidebar and start chatting!
